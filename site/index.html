복지 사각지대 모델링

교통망과 100m 격자 수요를 학습해 복지 사각지대를 계산하고, 신규 복지시설의 위치와 수용량을 제안하는 GNN·강화학습 기반 시설입지 방법론

문서 상태 설계 가설 · 실증 전

주요 독자 ML 엔지니어 · 공간 분석가 · 정책 기획자

01

기술 요약

권장 구조는 `GNN + 강화학습 + 기계적 수요 배분 평가기`입니다. ML이 시설 위치와 규모를 제안하고, 최적화 알고리즘이 실현 가능성과 정책 효과를 검증합니다.

  • 교통망은 이미지가 아니라 이동비용으로 변환합니다.도로·버스·지하철의 연결성, 대기, 환승, 보행 장벽을 격자–시설 OD 비용에 반영합니다.
  • GNN–RL이 주 의사결정 모델입니다.정답 위치 라벨 없이 미충족 인구, 이동비용, 설치비, 형평성을 보상으로 학습합니다.
  • Graph Diffusion은 대안 생성기로 제한합니다.하나의 정답보다 조건별 복수 배치안을 만들고 feasibility projection과 local search로 보정합니다.
  • 노인 인구와 실제 복지 수요를 구분합니다.필요율 데이터가 없다면 결과는 ‘복지 필요 인원’이 아니라 ‘공간적 접근 취약성’입니다.
02

핵심 결론: 학습기와 평가기의 책임을 분리합니다

시설입지는 이미지 생성보다 제약이 있는 조합 최적화에 가깝습니다. 아래 구조는 ML을 핵심에 두면서도 결과의 인원수와 제약 준수 여부를 설명할 수 있게 합니다.

R

= 추가 커버 인원 − α · 이동비용 − β · 설치비 − γ · 지역 간 불평등

03

출력은 사각지대 확률보다 미충족 인원으로 정의합니다

분석 단위는 100m 격자 i, 공급 단위는 복지시설 j입니다. 모델이 계산해야 할 결과는 배정되지 못한 수요 uᵢ와 커버 비율입니다.

해석 주의

독거 여부, 연령대, 장애, 소득, 차량 보유, 경사도, 실제 이용·대기·거절 기록이 없다면 결과를 “복지 필요 인원”으로 부르면 안 됩니다. 이 경우 분석 대상은 노인 인구 기준 공간적 접근 취약성입니다.

04

교통망을 이동비용으로 압축한 뒤 학습과 최적화를 연결합니다

도로·버스·지하철을 단순 이미지 채널로 쌓으면 연결성, 환승, 대기시간, 방향성을 잃을 수 있습니다. 먼저 격자–시설 OD 이동비용을 계산합니다.

Gij = Twalkij + Twaitij + Trideij + αNtransferij + βBij

Ntransfer
환승 횟수
B
경사·횡단·보행 단절 등 장벽
  1. 1
    공간 데이터 정렬좌표계·시간대·서비스 단위 통일
  2. 2

  4. 4

  5. 5

  6. 6
    보상·검증효과·비용·형평성 평가
05

모델은 위치·규모를 선택하고 최적화 레이어가 인원수를 계산합니다

정책 행동 aₜ = (설치 위치, 시설 규모)는 후보지를 하나씩 선택합니다. 수용량은 50·100·200명처럼 이산 클래스로 제한하는 편이 안정적이고 행정 시나리오와도 연결하기 쉽습니다.

Node

격자 피처 xᵢ

노인 인구, 현재 접근성, 대중교통 접근성, 주변 잔여 용량, 취약도, 설치 가능 여부·비용

Edge

이동시간 간선 eᵢⱼ

시간 임계값 또는 k-최근접 이동시간으로 연결하고, 도보·대중교통 비용과 환승을 보존

Action

위치·규모 선택 aₜ

선택된 시설 상태를 갱신하며 예산이나 시설 수가 소진될 때까지 순차적으로 행동

Evaluator

수요 배분 xᵢⱼ

최대 접근비용과 시설 용량을 지키면서 미충족 인원을 직접 계산

Σj xij + ui = Di

모든 수요는 배정되거나 미충족으로 남습니다.

Σi xij ≤ Cj

시설이 수용량을 넘겨 담당할 수 없습니다.

xij = 0 if Gij > Gmax

허용 이동비용 밖의 시설에는 배정하지 않습니다.
06

생성형 모델은 hard constraint를 직접 보장하지 못합니다

각 접근법을 배제하기보다 전체 시스템에서 책임을 명확히 나눕니다. Diffusion은 여러 Pareto 후보를 생성할 때 가치가 있지만, 첫 모델로 쓰려면 학습 데이터와 제약 보정 비용이 큽니다.

Graph Diffusion서로 다른 목적의 후보 배치를 다수 생성준최적 해 데이터와 feasibility 보정 필요Pareto 후보 생성기

기준선과 GNN–RL이 안정된 뒤, 비용 최소화형·미충족 인구 최소화형·형평성 우선형 배치안을 동시에 제시하려는 단계에서 추가합니다.

  • 최적화 솔버로 준최적 학습 해 생성
  • 예산·시설 수·설치 가능 지역을 조건으로 입력
  • feasibility projection + local search 후처리
07

검증은 정확도 대신 정책 효과와 강건성을 측정합니다

정답 위치 라벨이 없으므로 분류 정확도는 핵심 지표가 아닙니다. 작은 문제에서는 MILP 최적해와의 optimality gap을, 큰 문제에서는 휴리스틱 대비 목적함수와 실행시간을 비교합니다.

결론을 바꿀 수 있는 네 가지 불확실성

01

수요율

Pᵢ를 Dᵢ로 쓰면 모든 노인이 같은 필요도를 가진다고 가정합니다.

02

수용량

급식·상담·방문복지처럼 대체 불가능한 서비스 용량을 분리해야 합니다.

03

이동비용

배차, 대기, 환승, 보행 장벽과 시간대별 변화를 반영해야 합니다.

04

보상 가중치

최소 세 가지 정책 시나리오와 최악 지역 미충족 비율을 함께 보고해야 합니다.

08

기준선을 먼저 고정하고 GNN–RL을 단계적으로 검증합니다

ML이 반드시 핵심이어도 계산 가능한 기준선이 있어야 학습 효과와 실패 원인을 증명할 수 있습니다.

  1. Phase 1
    E2SFCA + 최소비용 흐름

    설명 가능한 접근성·미충족 기준선을 만들고 격자별 uᵢ 계산을 재현합니다.

    완료 조건 · 수요와 수용량 단위 검증
  2. Phase 2
    GNN + 강화학습

    동일한 그래프와 평가기에서 위치·규모 선택 정책을 학습합니다.

    완료 조건 · 휴리스틱 대비 효과·시간·제약 준수 개선
  3. Phase 3
    Graph Diffusion + local search

    조건별 복수 Pareto 대안을 만들고 제약을 후처리합니다.

    완료 조건 · 다양성·optimality gap·제약 위반률 검증

다음 설계 결정

  1. 분석 대상 복지서비스와 서비스별 수용량을 분리할 수 있는가?
  2. 수요의 분모는 전체 노인 인구인가, 예상 이용자 수인가, 우선 지원 대상인가?
  3. 최대 이동시간과 환승 횟수는 시간대·연령·서비스별로 어떻게 다른가?
  4. 후보지, 예산, 규모 클래스, 설치 불가능 지역은 어떤 행정 제약을 따르는가?
  5. 형평성은 최대 미충족 비율, 지역 간 분산, 취약도 가중 수요 중 무엇으로 정의할 것인가?

이 질문에 답하기 전의 결과는 정책 결정이 아니라 방법론 프로토타입으로 해석해야 합니다.
