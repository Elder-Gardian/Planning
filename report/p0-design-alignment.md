- 기준 설계: 현재 `main`의 `README.md`, `content/planning.md`, `content/technical-blueprint.md`, `content/dual-graph-recommendation.md`

- 판정 원칙: 충돌 시 현재 `main`의 명시적 P0 계약을 우선한다.

## 결론

인수본의 핵심 구조인 `전 후보 exact evaluation → GNN 순위학습 → top-k exact reranking`, 상태 확장 교통 그래프, 형평성 사전식 배분, reward hacking 방지 원칙은 현재 설계와 일치한다.

다만 인수본은 독립적인 최종 명세로 채택하지 않는다. 아래 충돌과 누락을 현재 설계 기준으로 교정한 뒤, 데이터 감사 결과와 1km 문맥 계층만 선택적으로 활용한다.

## 확정 판정

| 항목 | 판정 | P0 확정 계약 |
| --- | --- | --- |
| 대상 | 일치 | 대상 자치구에 도시지역 경로당 1개를 배치한다. |
| 수요 | 교정 필요 | 입력 `예상 동시 이용자 수`를 수요로 사용한다. `고령인구 × 참여율 × 동시체류율`은 직접 수요가 없을 때의 별도 추정 시나리오일 뿐이다. |
| 기존 용량 | 충돌 | 실제 동시수용량을 우선한다. 고정 8명은 운영 기준 공급에서 제거하고 불확실성 시나리오 필드로 대체한다. |
| 신규 용량 | 일치 | 예산·필지·법규로 연면적을 구한 뒤, 검증된 순사용면적/동시수용량 표본으로 용량을 파생한다. 법정 최소면적과 최소정원을 면적 밀도로 환산하지 않는다. |
| 행동 | 충돌 | 모델 행동은 `후보 100m 격자 1개`다. 필지와 사업안은 격자 내부의 결정론적 solver가 선택한다. |
| 중복 입지 | 일치 | 같은 격자에 기존 경로당이 있어도 별도 필지·건물·증축안이 실현 가능하면 허용한다. |
| 설치 가능성 | 일치 | 필지별 `FEASIBLE`, `HARD_EXCLUDE`, `REVIEW_REQUIRED`, `UNKNOWN` 상태를 사용한다. |
| 1km 계층 | 조건부 채택 | 100m만 수요·행동 노드다. 1km는 광역 문맥과 N/A 수요 보정에만 사용하고 수요에 중복 합산하지 않는다. |
| 교통 | 일치 | 노선·방향·정차순서·탑승상태를 보존하는 상태 확장 그래프가 도보·대기·차내·환승 비용을 계산한다. |
| 외부 경로 | 보강 필요 | 수요·후보·시설 이용자격·라우팅 마스크를 분리한다. 라우팅은 수도권 전체를 허용하고, 외부 시설은 이용자격과 시간 상한으로 제한한다. |
| 배분 목적 | 보강 필요 | `미수용 → CVaR90 → 선호시간 밖 사각지대 → 평균비용 → 사업비` 순서를 유지하고, 미수용을 CVaR 부담에 포함한다. |
| 시나리오 | 보강 필요 | 용량·수요·교통 시나리오의 기대값과 최악값을 함께 비교하는 사전식 `J_robust`를 사용한다. |
| Reward hacking | 일치 | 직접도보를 항상 후보로 포함하고 탑승 자체에는 보상하지 않는다. 모든 이동비용은 양수이며 루프·환승 은닉·수요 축소·용량 초과를 차단한다. |
| AI 역할 | 일치 | GNN은 후보 탐색 가속기다. 최종 제약 검증과 정책 순위는 exact solver가 담당한다. |

## 구현 불변식

1. 수요 입력과 평가 시나리오는 모델이 변경할 수 없다.
2. 총배정량과 미수용량의 합은 전체 수요와 일치한다.
3. 시설별 배정량은 어떤 시나리오에서도 용량을 넘지 않는다.
4. 직접도보 비용보다 불필요한 승차·환승 경로가 유리해질 수 없다.
5. 수요가 0인 격자는 CVaR·비율 계산에서 0으로 안전 처리한다.
6. 광운대역에서 석계역으로 이동해 6호선을 타는 경계 밖 경로가 연결된다.
7. 노원구 수요가 구로구 시설로 비현실적으로 배정되지 않도록 이용자격과 일반화비용 hard limit를 모두 적용한다.
8. 1km 인구는 100m 수요와 합산하지 않는다.
9. 고정 8명과 검증되지 않은 `1평당 1명` 값은 실제 동시수용량으로 노출하지 않는다.

## 선택적 이관 대상

- 100m/1km 격자 품질 진단과 경계 중복 처리 규칙
- 시설 주소·건축물대장·좌표 수집 파이프라인
- 용량의 관측값·대체값·신뢰도·시나리오 분리 개념
- 1km `parent_context` 계층

인수본의 기존 Pages, 강화학습 중심 문서, 고정 8명 운영 시나리오, 격자/필지 혼합 행동 정의는 이관하지 않는다.
