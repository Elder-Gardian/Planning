# 복지 사각지대 모델링 방법론 설계 노트

이 문서는 사용자가 제공한 ChatGPT 대화 「복지 사각지대 모델링」의 제안을 분석 보고서용 근거 노트로 정리한 것입니다. 외부 실증 데이터나 모델 성능 실험 결과가 아니라 **설계 가설과 권장 아키텍처**를 담고 있습니다.

## 문제 정의

- 100m 격자별 노인 인구, 도로·버스·지하철, 복지관 위치·수용량을 사용한다.
- 출력은 복지 사각지대 위치, 격자별 미충족 인원, 신규 시설의 위치·수용량이다.
- 노인 인구만 있을 때는 실제 복지 수요가 아니라 공간적 접근 취약성을 나타낸다.

## 핵심 방법론

1. 교통망을 이미지 채널로 직접 입력하지 않고 격자–시설 간 일반화 이동비용으로 변환한다.
2. E2SFCA와 용량 제약 최소비용 흐름을 설명 가능한 기준선으로 사용한다.
3. GNN이 교통·수요 그래프를 인코딩하고 강화학습 정책이 시설 위치·규모를 선택한다.
4. 최소비용 수요 배분 알고리즘이 미충족 인구, 이동비용, 비용, 형평성을 계산해 보상으로 반환한다.
5. Graph Diffusion은 조건별 복수 후보 배치를 생성하는 확장 모델로 사용하고 local search로 제약을 보정한다.
6. GAN은 정답 분포 부재와 hard constraint 문제로 우선순위가 낮다.

## 시각화 계약

- 분석 질문: 어떤 ML 접근법이 시설입지 의사결정의 핵심 모델에 가장 적합한가?
- 한 문장 결론: GNN–RL을 핵심 정책 학습기로, Graph Diffusion을 후보 생성기로 배치한다.
- 차트 유형: 긴 범주 이름을 비교하는 정렬된 가로 막대.
- 데이터 충분성: 네 개의 의도적으로 비교할 접근법이며 각 점은 정성 순서형 등급이다.
- 팔레트: 단일 blue root와 중립색. 색상 외에 정렬·값 라벨·텍스트 등급으로 구분한다.
- 주의: 등급은 실험 성능값이 아니며, 실제 배치 문제에서 별도 벤치마크가 필요하다.
